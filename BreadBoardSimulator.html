Breadboard Simulator

Breadboard Simulator is a virtual breadboard software developed for our semester project. It has an intuitive user interface that simulates a virtual breadboard and supports a wide range of components commonly used in college-level electronics, including:

Passives: Resistors (with correct colour bands)
Digital ICs: Basic logic Gates, Universal logic gates, Display Decoders and Parallel Adder
Outputs: LEDs, 7-segment Displays

These software provide a really good environment for designing various circuits, where you simply drag and drop the circuit components and then simulate them. Voltages and currents can also be seen by hovering over pins. Furthermore, a circuit can be simply saved for future references.
